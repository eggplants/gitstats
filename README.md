# gitstats for python 3

Fork from <https://github.com/hoxu/gitstats> for support python 3

## Install

```bash
pip install git+https://github.com/KaivnD/gitstats
```

## Usage

```shellsession
$ gitstats -h
```

## For Example

``` bash
gitstats -c start_date=2021-07-14 <gitdir> <outputdir>
```
